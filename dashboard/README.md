# Admin Dashboard

A modern, feature-rich admin dashboard built with Next.js 16 and Tailwind CSS.

## Features

✅ **Responsive Sidebar Navigation**
- Fixed left sidebar with smooth transitions
- Mobile-responsive with hamburger menu
- Active route highlighting
- User profile section

✅ **Dashboard Pages**
- **Overview Dashboard**: Stats, metrics, charts, and recent activity
- **Users Management**: User list, search, filters, and pagination
- **Jobs Management**: Job listings, stats, and management tools
- **Analytics & Reports**: Performance metrics and insights
- **Settings**: General, notifications, security, and profile settings

✅ **Modern UI Components**
- Gradient cards and badges
- Interactive tables
- Stats cards with icons
- Toggle switches
- Search and filter controls
- Pagination

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3001](http://localhost:3001) in your browser

## Routes

- `/` - Landing page with auto-redirect
- `/dashboard` - Main dashboard overview
- `/dashboard/users` - Users management
- `/dashboard/jobs` - Jobs management
- `/dashboard/analytics` - Analytics & reports
- `/dashboard/settings` - Settings page
